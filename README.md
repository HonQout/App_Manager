# App Manager
## Basic information
This is an app which can provide fundamental information about all apps installed on your device. 
## Last updated: 2025-11-26
### Changes and improvements:
1. Altered the implementation of DiffUtils.Callback in some RecyclerView adapters.
2. Added dialogs as click callback to the items of RecyclerView of every scope of package information, presenting more information about each item.
## Function
1. Have a better understanding of all apps installed on your device.
2. Launch any activity of these apps.
